Outils

PhpMyAdmin

PhpMyAdmin est une interface graphique permettant de gérer des bases de données.

Cette interface est réalisée en PHP et est accessible depuis un navigateur web.

http://localhost/phpmyadmin

MySQL Workbench

MySQL Workbench est un logiciel (pas une interface web) permettant d’effectuer des actions similaires à PhpMyAdmin.

Il est cependant plus pratique pour effectuer des requêtes (coloration syntaxique, auto-complétion, onglets...) et permet également de réaliser un schéma de base de données pour ensuite générer un script SQL.

Paramétrer MySQL Workbench

Aller dans les préférence du logiciel puis :

  • "SQL Editor" : Décocher la case "Safe Updates" (il s'agit d'une sécurité permettant de bloquer l'execution des requêtes UPDATE ou DELETE sans clause WHERE)
  • "SQL Editor" > "Query Editor" : Cocher la case "Use UPPERCASE keywords on completion"
  • "Modeling" > "Defaults" :
    • "PK Column name" : id
    • "PK Column Type" : BIGINT
    • "Column Type" : VARCHAR(255)

Redémarrer ensuite le logiciel pour appliquer les paramètres.
