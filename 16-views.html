Les vues

Créer une vue

Les vues sont des tables virtuelles générées à partir d'une requête SELECT.

Une vue peut donc retourner des données traitées et provenant de tables différentes.

CREATE OR REPLACE VIEW utilisateur_extended AS
SELECT *,
    CONCAT(nom, ' ', prenom) AS fullname,
    DATE_FORMAT(date_naissance, '%d %M %Y') AS date_naissance_format
FROM utilisateur

Utiliser une vue

Les vues s'utilisent comme des tables classiques :

SELECT * FROM utilisateur_extended;

Supprimer une vue

Pour supprimer une vue, vous pouvez utiliser la l'instruction ci-dessous :

DROP VIEW utilisateur_extended;
