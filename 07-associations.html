Les associations

1-1 (one-to-one)

Rarement utilisée. Permet d’étendre les informations contenues dans une table sans la surcharger (informations complémentaires sur une personne par exemple).

Permet également de mettre en place un système d'héritage (POO) dans une base de données.

1-1 (one-to-one)

1-N (one-to-many)

Permet d'associer une entité à plusieurs entités.

N-N (many-to-many)

Permet d'associer plusieurs entités à plusieurs entités.
